Сценарий: «Автоматическое создание дерева сертификатов и личного сертификата для нового сотрудника, создание транспортного контейнера PKCS, автоматическая загрузка его в хранилище средствами Windows»

Скрипт создания сертификата с использованием openssl из работы 2 должен производить сл. действия:
1. В конечный сертификат (см из предыдущее задание) добавляется собственное расширение (назначение и название -- на усмотрение студента)
2. Все 3 сертификата (root, mca, user) и закрытый ключ конечного сертификата собираются в 1 пакетный файл формата .p12 (PKCS#12)
3. Экспорт закрытого ключа из .p12 (PKCS#12) должен быть запрещен
4. Файл .p12 автоматически импортируется в хранилище Windows (используйте утилиту Certmgr.exe или аналог)

Таким образом, при запуске скрипта должно создаваться дерево сертификатов, удовлетворяющее все перечисленным условиям и автоматически импортироваться в хранилище Windows.
